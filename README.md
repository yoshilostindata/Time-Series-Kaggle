# Time Series Forecasting — Kaggle Crypto

## Описание проекта

В данном проекте решается задача прогнозирования финансовых временных рядов криптоактивов в рамках Kaggle-соревнования.

## Задача

Цель — предсказать целевую величину (доходность/цену актива) на следующий момент времени на основе исторических данных.  
Метрика качества — корреляция или регрессионная метрика (в зависимости от постановки соревнования).

## Подход

Модель работает с многомерными временными рядами:

- загружаются и анализируются данные (`train.csv`, `asset_details.csv`);
- выполняется предобработка и нормализация признаков;
- формируются окна фиксированной длины (sliding window);
- входные последовательности подаются в нейронную сеть;
- модель предсказывает значение целевой переменной на следующий шаг.

## Архитектура модели

- нейронная сеть на PyTorch;
- последовательная обработка временных данных (RNN/LSTM или полносвязная модель поверх окон признаков);
- функция потерь — MSE;
- оптимизатор — Adam.

## Особенности

- учёт временной структуры данных;
- предотвращение утечки информации из будущего;
- разделение на train/validation;
- подготовка submission-файла для Kaggle.

## Итог

Проект демонстрирует применение глубокого обучения для финансового time series forecasting с корректной обработкой временной зависимости и оценкой качества модели.
