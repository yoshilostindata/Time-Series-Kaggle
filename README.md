В этом проекте решается задача прогнозирования временных рядов криптоактивов (Kaggle Crypto Time Series).


Данные представляют собой многомерные временные ряды по разным активам.
После предобработки формируются последовательности фиксированной длины (окна), которые подаются в модель для предсказания целевой переменной на следующий момент времени.


Используется нейронная сеть на PyTorch, ориентированная на работу с последовательностями:

вход: окно из нескольких предыдущих таймстемпов с признаками актива;

архитектура: LSTM

выход: числовое предсказание целевой величины (регрессия);

функция потерь: MSE;

оптимизация: Adam.

Модель обучается на исторических данных, после чего используется для генерации предсказаний и формирования submission-файла для Kaggle.

Основная цель — корректно учитывать временную структуру данных и избежать утечки информации из будущего.
